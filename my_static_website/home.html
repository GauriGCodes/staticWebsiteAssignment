Hello, I'm Gauri Gupta.....

I started my Masters in Computer Science at Pace in Spring 2023 and this Semester I have registered for below 3 course

CS608 - Algorithms and Computing Theory

In Algorithms and Computing theory, I am learning methods for expressing and comparing algorithm complexity (worst- and average-case upper bounds, lower bounds) as well as to verify correctness.Moreover, I am learning Algorithm-design techniques (divide-and-conquer, dynamic programming) as well as data structures (trees, heaps, hash tables) widely used in modern software development. The knowledge gained will be applied to a variety of practical problems, such as searching, sorting, and graph problems (shortest paths, minimum spanning trees).

CS612 - Concepts and Structures: Internet Computing

In Internet Computing, I am learning fundamental Concepts of software framework, cluster based, cloud computing and distributed computing. Moreover, I am understanding various web development technologies: CSS, HTML5 by implementing static web application and web application using ReactJS. Also, I am learning about technical challenges of building responsive web applications and how these challenges are addressed using javascript frameworks ReactJs and AngularJS;

CS623 - Database Management Systems

The focus of the course is on database management system installation and configuration, and database role as a middleware in system hierarchy, entity relationship model for logical design, schema normalization and performance tradeoffs, database management with SQL through database console, database programming through JDBC, event- processing with triggers, efficient data processing with stored-procedures, transactions management and ACID properties, database security, and crash recovery.

My favorite class is Internet Computing as it allows me to explore my passion of web development in an interactive and fun manner
